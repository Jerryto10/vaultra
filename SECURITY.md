# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅ Yes    |

## Reporting a Vulnerability

Vaultra takes security seriously. If you discover a vulnerability, please **do not** open a public GitHub issue.

Instead, report it privately to: **security@vaultra.io**

We will respond within 48 hours and coordinate a fix before public disclosure.
